## API Reference

#### Add and list Repositories

```http
  GET /
```

#### Details of the repository

```http
  GET /repository/[name]
```

| Parameter | Type     | Description                         |
| :-------- | :------- | :---------------------------------- |
| `name`    | `string` | **Required**. Name of item to fetch |
